My First Post

Update Feb 26, 2026 • 3 min read

This is a placeholder post so I can test my layout, theme toggle, and overall blog structure.

What this site is for

I want a calm, minimal space to post updates. Each post is just an HTML file in the “posts” folder.

How I’ll add new posts

I’ll duplicate the post-template file, rename it, and then update the title, date, tag, and content.

Quick checklist

  • Duplicate posts/post-template.html
  • Rename to something like posts/my-new-post.html
  • Update the title, date, and content
  • Add a new card link on the homepage
